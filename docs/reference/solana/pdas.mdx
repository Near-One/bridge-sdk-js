<Warning>
  PDA seeds must match the on-chain program exactly. Always use the builder's derive methods rather than computing PDAs manually.
</Warning>

## Derive Methods

### deriveConfig

Gets the bridge configuration PDA.

```typescript
const configPda = solana.deriveConfig()
```

### deriveAuthority

Gets the program authority PDA.

```typescript
const authorityPda = solana.deriveAuthority()
```

### deriveVault

Gets the token vault PDA for a mint.

```typescript
const vaultPda = solana.deriveVault(mintPublicKey)
```

### deriveSolVault

Gets the native SOL vault PDA.

```typescript
const solVaultPda = solana.deriveSolVault()
```

### deriveWrappedMint

Gets the wrapped token mint PDA for a token from another chain.

```typescript
const wrappedMint = solana.deriveWrappedMint("near:wrap.near")
```

## Example Usage

```typescript
import { createSolanaBuilder } from "@omni-bridge/solana"
import { PublicKey } from "@solana/web3.js"

const solana = createSolanaBuilder({ network: "mainnet" })

// Get all relevant PDAs
const config = solana.deriveConfig()
const authority = solana.deriveAuthority()
const solVault = solana.deriveSolVault()

// For a specific token
const usdcMint = new PublicKey("EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v")
const usdcVault = solana.deriveVault(usdcMint)

console.log({
  config: config.toBase58(),
  authority: authority.toBase58(),
  solVault: solVault.toBase58(),
  usdcVault: usdcVault.toBase58(),
})
```

## PDA Seeds

The PDAs use these seeds (for reference only - use the derive methods):

| PDA | Seeds |
|-----|-------|
| Config | `["config"]` |
| Authority | `["authority"]` |
| SOL Vault | `["sol_vault"]` |
| Token Vault | `["vault", mint]` |
| Wrapped Mint | `["wrapped_mint", token_address]` |
